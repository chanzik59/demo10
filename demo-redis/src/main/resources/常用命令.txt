通用：
keys  查询所有的key
type  key的类型
ttl  返回key的剩余过期时间 秒
expire 设置过期时间秒
persist 移除过期时间
rename 重命名
exists 判断是否存在
move 移动到其他库
del 删除

字符串：
set  新增字符串
setNx 新增字符串如果不存在
setEx 新增字符串并且设置超时时间
setRange 替换内容
mSet  批量新增字符串
get 获取字符串
getRange 截取字符串
mGet 批量获取字符串
strLen 获取字符串长度
append 追加内容
incr 加一
incrBy 自定义加
incrByFloat 加上一个浮点数
decr 减一
decrBy 减去任意数

哈希值：
hGet 获取字段值
hmGet 批量获取字段值
hGetAll 获取所有键值
hSet 新增字段
hSetNx 如果不存在设置字段值
hmSet 批量设置字段值
hKeys 获取所有字段
hVals 获所有值
hExists 判断字段是否存在
hIncrBy 字段值加上整形
hIncrByFloat  字段加上浮点
hDel 删除字段
hLen 字段长度

列表：
lPush 从左边插入一个元素
lPushX 如果列表存在左边插入一个元素
rPush 从右边插入一个元素
rPushX 如果元素存在右边插入一个元素
bLPop 列表左边弹出一个元素 可以阻塞
bRPop 列表右边弹出一个元素 可以阻塞
lInsert 在某个元素前或者后插入一个元素
lTrim 对列表进行截取
lLen 统计列表长度
lIndex 通过索引获取元素
lRem 删除元素
lRange 获取范围内的元素
